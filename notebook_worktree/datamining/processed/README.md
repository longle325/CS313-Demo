# Processed Data

Processed 2009-2024 tables used directly by `FINAL_FULL_PIPELINE_CS313_BIODIVERSITY.ipynb`.

- `gbif_diversity_2009_2024.csv`: biodiversity labels and effort-adjusted diversity metrics by `grid_id, year`.
- `forest_stats_2009_2024.csv`: Hansen forest cover and forest loss by `grid_id, year`.
- `weather_yearly_2009_2024.csv`: Open-Meteo annual temperature/rainfall summaries by `grid_id, year`.
